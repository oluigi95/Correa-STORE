# Correa Store

Projeto de uma landing page moderna para apresentação de produtos de tecnologia.

O site apresenta um carrossel animado de produtos com transições suaves, indicadores de navegação e design futurista inspirado em lojas de tecnologia.

## 🚀 Funcionalidades

- Carrossel de produtos com animação
- Navegação com botões (anterior / próximo)
- Indicadores visuais de produto ativo
- Troca automática de produtos a cada 5 segundos
- Layout moderno com animações em CSS
- Design responsivo

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar:

- Manipulação do DOM com JavaScript
- Criação de animações com CSS
- Estruturação de layouts modernos com HTML e CSS
- Lógica de carrossel de produtos
